# 摆摊助手

> 边做边卖、边接单、边收款、叫号不用喊

## 项目概述

摆摊助手是一款为线下小吃摊主设计的全流程效率工具，帮助摊主解决边制作边接单、收款、对账的痛点。

## 产品形态

| 端 | 技术栈 | 访问方式 |
|---|---|---|
| 摊主端APP | React + Web | http://localhost:3000 |
| 顾客端 | HTML + JS | 直接打开 `customer-app/index.html` |

## 功能特性

### 摊主端
- 📋 实时订单列表，支持按状态筛选
- 🔔 语音播报新订单（支持息屏播报）
- 👆 一键接单、完成、叫号
- 📊 今日订单数、营收统计
- 📺 叫号大屏（屏幕+语音）

### 顾客端
- 🏪 扫码即用，无需下载
- 🎤 语音点餐（模拟）
- 📱 微信/支付宝支付
- 🔔 取餐进度通知
- 📋 订单状态实时查看

## 核心页面

### 摊主端
1. **订单主页** - 实时订单列表，支持接单/完成/叫号
2. **统计卡片** - 今日订单数、营收汇总
3. **叫号大屏** - 全屏显示取餐号

### 顾客端
1. **商品页** - 商品列表、加入购物车
2. **规格弹窗** - 选择大小、加料、口味
3. **订单页** - 取餐号、等待进度

## 技术亮点

- 🎨 极简设计，大字体、大按钮，单手操作
- 🔊 语音合成 (TTS) 实现订单播报
- 📱 移动端优先，完美适配手机
- ⚡ 实时状态更新，模拟真实业务场景

## 演示说明

### 摊主端演示
1. 打开 http://localhost:3000
2. 点击"测试新订单"按钮模拟新订单
3. 语音会自动播放"新订单，A0X号，xxx"
4. 点击"接单"开始制作
5. 点击"完成并叫号"触发叫号大屏

### 顾客端演示
1. 打开 `customer-app/index.html`
2. 选择商品加入购物车
3. 点击"去结算"生成取餐号
4. 等待15秒会弹出取餐提醒
